# إمكانية الوصول (WCAG 2.1) — BauPass

## الهدف

استيفاء متطلبات الجهات الحكومية والمؤسسات العامة (مستوى **AA** كهدف افتراضي).

## نطاق المرحلة الأولى

| وحدة | أولوية |
|------|--------|
| `admin-v2` | عالية |
| `index.html` (legacy admin) | متوسطة |
| `emp-app.html` (عامل) | عالية |
| تقارير PDF | نصوص قابلة للقراءة + تباين |

## معايير عملية

- تباين لوني ≥ 4.5:1 للنص العادي
- كل عنصر تفاعلي: `focus-visible` و `aria-label` / `aria-labelledby`
- نماذج: ربط `<label>` بـ `for` / `id`
- لا اعتماد على اللون وحده للحالة (أيقونة + نص)
- دعم `prefers-reduced-motion`
- لغة الصفحة: `lang` و `dir` (موجود جزئياً)

## فحص

- axe DevTools على admin-v2 و emp-app
- لوحة مفاتيح فقط: تسجيل دخول · inbox · عامل check-in
- CI (مستقبل): Playwright + `@axe-core/playwright` على smoke paths

## استثناءات مؤقتة

- خرائط Leaflet: بديل قائمة للمواقع
- محررات غنية: نص بديل للعمليات الحرجة
