# Smoke-Check nach Deploy (Calls / Legal / Tausch)

Kurz prüfen nach Backend-Deploy und neuer APK:

## Anrufe
- [ ] AG → MA 1:1 Audio (annehmen/ablehnen)
- [ ] MA → AG 1:1 Audio
- [ ] Konferenz: Einladung mit geschlossener APK (Push/CallKit)
- [ ] Kamera-Hinweis sichtbar; AG und MA sehen sich bei Cam an
- [ ] ICE-Zeile im MA-Overlay bei verbundenem Anruf (ICE/Pfad)

## Schichttausch
- [ ] Tausch vorschlagen → Push öffnet Tab „Tausch“
- [ ] Annehmen / Ablehnen
- [ ] Gesendete Anfrage zurückziehen
- [ ] Verlauf zeigt Status

## Rechtliches
- [ ] Login-APK: Impressum & Datenschutz lesbar
- [ ] Profil → Rechtliches (Firmatexte)
- [ ] Datenschutz-Vorlage im Admin speichern
- [ ] Consent erneut, wenn Text geändert (Version)
- [ ] Auskunft / Löschung Anfrage aus der App

## Chat
- [ ] Sprachnotiz E2E (kein Klartext „Sprachnachricht“)
- [ ] Bildanhang senden/empfangen

## Assistant
- [ ] Ohne API-Key: Offline-Hinweise (Check-in/Urlaub/Schicht)
- [ ] Mit API-Key: normale Antworten

## Store
- [ ] `https://DOMAIN/privacy.html` erreichbar
- [ ] `https://DOMAIN/impressum.html` erreichbar
