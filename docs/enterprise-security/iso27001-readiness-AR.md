# الاستعداد لـ ISO 27001 — WorkPass

## نطاق ISMS المقترح

منصة WorkPass SaaS / private cloud: هوية، دخول، بيانات عمال، مستندات، تقارير، فوترة.

## خريطة ضوابط (ملخص)

| Annex A (ISO 27001:2022) | WorkPass اليوم | الفجوة / العمل |
|--------------------------|---------------|----------------|
| A.5 سياسات | 🟡 وثائق `enterprise-security/` | اعتماد رسمي + مراجعة سنوية |
| A.8 إدارة الأصول | 🟡 تصنيف بيانات | تطبيق labels في DB/UI |
| A.9 تحكم بالدخول | ✅ RBAC + SSO جزئي | SAML كامل + أدوار مؤسسية |
| A.12 التشغيل | ✅ نسخ احتياطي | DR drill موثّق |
| A.16 حوادث | 🟡 IR plan | تمارين ربع سنوية |
| A.18 امتثال | 🟡 GDPR | DPA قالب للعملاء |

## أدلة مطلوبة للتدقيق

1. سياسة أمن معلومات (معتمدة من الإدارة)
2. سجل مخاطر + معالجة
3. سجل أصول (DB, buckets, secrets)
4. إجراءات onboarding/offboarding مستخدمين
5. سجلات تدقيق + احتفاظ
6. نتائج pentest
