# Data Governance

## تصنيف البيانات

انظر `enterprise-security/data-classification-AR.md`.

## دورة الحياة

1. **جمع** — حد أدنى من PII
2. **استخدام** — غرض تعاقدي (دخول/امتثال)
3. **احتفاظ** — سياسة per tenant
4. **حذف** — عند انتهاء العقد (مع استثناءات قانونية)
5. **تصدير** — DSR عند الطلب

## الموافقات

- موافقة العامل على معالجة البيانات (worker app)
- موافقة المستأجر على subprocessors (DPA)

|----------|--------|
| بيانات العمال | العميل (controller) |
| سجلات المنصة | BauPass (processor) |
