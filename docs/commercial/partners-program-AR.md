# برنامج الشركاء (Integrators & Resellers)

## أنواع الشركاء

| نوع | دور |
|-----|-----|
| Integrator | تركيب، SSO، تكاملات SAP/DATEV |
| Reseller | بيع اشتراكات · onboarding |
| OEM | White-label (مستقبل) |

## متطلبات الشريك

- شهادة منتج (تدريب 2 يوم)
- بيئة UAT
- اتفاقية DPA فرعية

## تقني

- حساب `partner-admin` (مستقبل) — حالياً superadmin لكل شريك
- API keys محدودة: `platform/api_platform/`

## تجاري

لا يُفعّل قبل استقرار المرحلة A.
