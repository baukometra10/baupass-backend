# BauPass — بدء سريع (صفحة واحدة)

## 1) دخول الإدارة

`https://baupass-production.up.railway.app/admin`

## 2) موظف + NFC

تبويب **الموظفون** → UID البطاقة → **حفظ** → **QR تفعيل**

## 3) تثبيت التطبيق

الموظف يمسح QR → APK أو TestFlight → تسجيل دخول مرة واحدة

## 4) حضور على الموقع

فتح التطبيق → **تسجيل دخول** (NFC أو GPS)

## 5) التحقق

تبويب **الحضور** أو Legacy → آخر عمليات الدخول

## مؤسسي (اختياري)

- **🏛 المؤسسة** — 16 طبقة  
- **Geofence · أتمتة · تكامل** — خريطة + SAP/M365  
- **المنصة** — AI (Enterprise + OpenAI)
