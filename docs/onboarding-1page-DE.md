# BauPass — Schnellstart (1 Seite)

## 1. Admin anmelden

`https://IHR-SERVICE.up.railway.app/admin`  
Superadmin oder Firmen-Admin.

## 2. Mitarbeiter + NFC

Tab **Mitarbeiter** → NFC-UID eintragen → **Speichern**  
Button **QR Aktivierung** → Link/QR an den Mitarbeiter.

## 3. App installieren

Mitarbeiter scannt QR → APK oder TestFlight → einmal anmelden.

## 4. Check-in am Bauzaun

App öffnen → **Check-in** (NFC oder Standort, je nach Modus).

## 5. Kontrolle

Tab **Anwesenheit** oder Legacy **Zutritt** — letzter Eintritt sichtbar.

## Enterprise (optional)

- **🏛 Enterprise** — 16 Plattform-Schichten  
- **Geofence · Auto · Integration** — Karte, Regeln, SAP/M365  
- **Plattform** — AI (Enterprise + OpenAI Key)
