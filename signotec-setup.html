Signotec — Einmalige Einrichtung

Nur auf diesem Windows-PC · einmal pro Rechner · USB-Pad anschließen

Ihr WorkPass-Login (Admin oder Firma) hat keinen Einfluss. Signotec läuft lokal auf diesem PC, nicht auf dem Server.

1. Setup-Datei herunterladen
  1. Datei speichern — baupass-signotec-setup.bat (z. B. Downloads)
  2. Rechtsklick auf die Datei → Als Administrator ausführen
  3. Im Installations-Assistenten Localhost wählen (nicht Default) → Installation abschließen
  4. Browser öffnet https://localhost:49494 → Erweitert → Risiko akzeptieren
  5. WorkPass neu laden (Strg+Shift+R) → Mitarbeiter → Signaturgerät

Das Setup lädt Signotec, startet den Dienst, öffnet die Zertifikatsseite und richtet Autostart ein. Log: Desktop baupass-signotec-setup.log

Signotec — One-time setup

On this Windows PC only · once per computer · connect USB pad

Your WorkPass login (admin or company) does not affect Signotec. It runs locally on this PC, not on the server.

1. Download setup file
  1. Save the file — baupass-signotec-setup.bat (e.g. Downloads)
  2. Right-click the file → Run as administrator
  3. In the installer wizard choose Localhost (not Default) → finish installation
  4. Browser opens https://localhost:49494 → Advanced → Accept the risk
  5. Reload WorkPass (Ctrl+Shift+R) → Workers → Signature device

Setup downloads Signotec, starts the service, opens the certificate page, and enables autostart. Log: Desktop baupass-signotec-setup.log

Signotec — إعداد لمرة واحدة

على هذا الكمبيوتر Windows فقط · مرة واحدة لكل جهاز · وصّل لوحة USB

تسجيل الدخول في WorkPass (مسؤول أو شركة) لا يؤثر. Signotec يعمل محلياً على هذا الجهاز وليس على السيرفر.

1. تحميل ملف الإعداد
  1. احفظ الملف — baupass-signotec-setup.bat (مثلاً في التنزيلات)
  2. كليك يمين على الملف → تشغيل كمسؤول (Run as administrator)
  3. في معالج التثبيت اختر Localhost (وليس Default) → أكمل التثبيت
  4. يفتح المتصفح https://localhost:49494 → متقدم → قبول المخاطرة
  5. أعد تحميل WorkPass (Ctrl+Shift+R) → الموظفون → جهاز التوقيع

الإعداد يحمّل Signotec ويشغّل الخدمة ويفتح صفحة الشهادة ويُفعّل التشغيل التلقائي. السجل: سطح المكتب baupass-signotec-setup.log
